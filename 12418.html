Windows XP将退 或将带动下一波换机潮

2009-02-12 20:00:35 作者：admin 新华网 浏览次数：0 网友评论 0 条 [评论][去论坛交流]

图片1：windows 7桌面

为了给即将上市Window 7让路，微软打算在10个星期之内把已经推出7年时间的Windows XP操作系统转入只提供有限的技术支持，从而开始淘汰Windows XP的过程。

市场预计，微软在今年5月底将正式停售版本(COEM)，版本，随著2008年6月微软终止与硬件OEM合作并停止供应零售盒装版本后，至此XP已经接近步入历史阶段。

目前，微软确认，用户可以以折扣价将原有的Windows XP升级至Windows 7，前提是用户需要全新安装Windows 7。微软没有透露XP升级的具体价格。分析认为，Windows 7的对硬件的要求远高于XP，目前按装XP的PC硬件很多可能并不能流畅运行经Windows 7，而随着Windows 7上市，将带动下一波换机潮。

只提供有限技术支持

据悉，按照微软的计划，Windows XP将在4月14日离开微软的“主流技术支持”计划进入“延长的技术支持”计划。一般来说，微软将产品在主流技术支持计划中保留五年时间，然后在延长的技术支持计划中保持五年。然而，由于Windows XP和它的后续产品Windows Vista之间的发布间隔时间太长，迫使微软把Windows XP技术支持的最后截止时间推迟到了13年。

两年前，微软将Windows XP家庭版和XP媒体中心版的主流技术支持的截止时间定为2009年和2014年。这些日期符合微软以前为Windows XP专业版规定的日期。

根据微软的政策，主流技术支持向每一个用户提供免费的补丁，包括安全补丁和其它修复瑕疵的补丁。在延长的技术支持阶段，所有的用户都可以得到安全补丁，但是，非安全性的热门补丁仅提供给与微软签署技术支持合同的企业。

不过，用户可在延长的技术支持阶段，通过付费获得技术支持，安全技术支持更新不需要支付额外的费用，热门补丁补丁是付费的。企业必须要在Windows XP在4月份退出主流技术支持阶段的90天之内购买延长的技术支持合同。

为Windows 7让路

Windows XP的技术支持服务进入“延长”，表明其“服役”期即将进入尾声将。

Windows XP是微软迄今为止最成功的操作系统。据研究机构Net Applications数据显示，自2001年10月上市，至2008年12月为止，XP在全球PC操作系统市占率达65.22%，而2007年1月问世的Windows Vista则为21.12%，虽然微软积极推动Vista，不过XP明显仍稳居市场龙头。

尽管微软一直力推Vista，但由于Vista上市之初因为兼容性问题，导致市场接受度不高也未带动换机潮，反而是XP停售期限一延再延；原本Vista上市后，微软已定调2008年1月将停止XP与OEM业者的合作搭售，不过因为Vista的市场反应不佳，因此跟著延后到2008年6 月，在此同时则因应Netbook、Nettop产品兴起，XP正式进入低价市场，成为微软抢攻低价计算机商机的主力操作系统产品。

微软周二公布了Windows7的6大版本，分别为Windows7初级版(Starter)、Windows7家庭基础版 (HomeBasic)、Windows7家庭高级版(HomePremium)、Windows7专业版(Professional)、 Windows7企业版(Enterprise)和Windows7旗舰版(Ultimate)。

目前，微软已确认，用户可以以折扣价将原有的Windows XP升级至Windows 7，前提是用户需要全新安装Windows 7。不过，目前微软还没有透露XP升级的具体价格。

分析认为，Windows 7的对硬件的要求远高于XP，目前按装XP的PC硬件很多可能并不能流畅运行经Windows 7，而随着Windows 7上市，将带动下一波换机潮。
