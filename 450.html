流浪儿童心声

时间:2006/11/13 栏目:关注流浪儿童 来源：英国救助儿童会 字体: 小 中 大

本文标签： 流浪儿童

心声－某地区流浪儿童救助保护中心
－　担心肚子饿
－　担心肚子饿时到哪儿找吃的
－　担心被别人打
－　不愿父母找到自己
－　长大了想当警察的头，能当多大就当多大。
－　建议修一所大房子，可容纳所有流浪的、没家的孩子。有书看、有电视、有电脑、可以学习炒菜和其他的家务；可以去读书上学；可以睡觉；可以吃东西；应该想去就去，想走就走……

苦恼－某市盲哑学校：
－　人行道上没有很多盲道，而且摆摊的不少，容易撞上去。
－　在十字路口时不知道红绿灯，很多时候是跟着别人走，还有就是听车子发动机的声音。
－　坐公共汽车时不知道是哪一路，到哪里？有的车报站名，有的没有。
－　希望能够订阅《盲童文学》。

建议－云南省少数民族地区学校：
－　教室里有些门、窗的锁应该换一下，因为它们随便一碰就坏了。
－　老师不准打学生，只可以说服他们。
－　我们想在学校种一些果树，如果哪个同学表现好，可以给他一个水果，当作奖励。
－　上课时还可以做游戏，又可以学又可以玩。
－　老师上课时脸上要有笑容。

话语－云南少数民族地区学校：
－　最喜欢听的课是数学课、音乐课和自然课。
－　老师常常提问学习差的学生。
－　作业太多，没按时完成，老师有时会骂。
－　有时课堂上很乱，老师讲的课没有吸引力。
－　老师组织我们捐钱捐粮帮助困难的同学。
－　在家帮助家长喂猪、喂鸡、抬土、煮饭、洗碗、扫地。
－　我帮我家的老母鸡孵小鸡。

儿童法制论坛全体参与者

"儿童法制论坛"宣言
行动，　让人人知法守法

很幸运，世界有了法律，自从有了它，它就寓于我们的生活，给予我们保护，维护我们的权益……
法像一把双刃剑，执法者用它惩治邪恶，每一位合法公民，用它来维护自己的权利；法像一张无形的网，紧密地联系着你，我，他；法像一面镜子，让人人在法律面前看清自己；法象一座天平，法律面前人人平等。
面对法律，我们承诺: 用心学法，懂法；尽心宣传，使人人守法、用法；让我们拉紧同伴的手，共同融入法律的时空，把法律的尊严延续，把法律的光芒散播。

参加联合国《儿童权利公约》培训的孩子认为他们的权利在生活中常常得不到尊重：
－　城市建筑师应该建造更多的公园，因为我们常常得穿过大街，这是非常危险的。
－　因为我来自农村，住在自己搭建的小窝棚里，在我遭到不幸或惹了麻烦时，往往得不到公平的对待。
－　我曾两次入狱……我还太年轻，不应将我关在那里。
－　当我的意见和大人不一致时，他们会说你还顶嘴，然后特别生气。
－　父母说家里没有多的钱，我不需要去上学，因为我是女孩子，但我想到学校去……

参加合肥第二届儿童论坛的孩子们心目中的好老师应该是：
－　思想开明
－　尊重同学，不打骂学生。
－　不拖堂，不过多占用学生休息时间。
－　能认真听取学生的意见和建议，多关心学生的学习及生活。
－　不在教室里吸烟
－　讲课声音洪亮，吐字清晰。
－　和蔼可亲，责任心强
－　不布置过多的作业
－　讲一口流利的普通话
－　公私分明，不以分数为标准划分好、差生，不歧视学生。
－　是同学的良师益友，多与学生沟通，多关心学生的学习及生活。

合肥市中学生在联合国《儿童权利公约》及中国相关法律知识竞赛上发出呼吁：

这个世界上,我们是幸福的孩子,我们有温暖的家，有亲爱的爸爸妈妈.我们可以尽情地玩耍,我们有敬爱的老师,可爱的同学,我们有自己喜爱的洋娃娃.我们不 再孤独,不再害怕.当我们无助时,我们有亲人,当我们失意时,我们有朋友.我们幸福地生活,成长在这样一个美好的时代，我们拥有儿童特有的权利。

我们都知道儿童是社会中最单纯、快乐的一群,可是他们的健康成长,应有的权利却并没有得到充分的保障.让我们每一个人都来关心这个问题,让我们伸出双手，用自己的热情与希望去奋斗,共筑一个美好的明天!

各位领导,各位老师，各位同学,今天，我们欢聚一堂,来参加本次<<儿童权利公约＞竞赛活动，就是希望所有的儿童更多,更好地了解我们儿童拥 有哪些权利,并且学会怎样去保护我们的权利,也希望大家将自己的收获和我们的朋友,同学一起分享,希望大家都能够度过一段快乐的时光! 谢谢！

关于儿童虐待和忽视，孩子们这样说：
－　不尊重孩子，打骂孩子，让孩子做办不到的事。
－　体罚学生，不完成作业，在脸上刻字。
－　犯错误抓住后，被打。
－　父母用烟头烫孩子
－　没有卖完鸡蛋，家长打骂孩子，不让回家。
－　小孩偷了钱，家长发现后，打碎了瓶子，让跪在玻璃上。
－　父母对孩子说："滚，我不养你了。"

孩子们谈到体罚之外的教育方式：
－　讲故事，说明所犯的错误。
－　给孩子一定的游戏时间。
－　孩子犯了错误，家长应用幽默的语气来点明孩子的错误。
－　让孩子当一次家长，家长也当一次孩子，也做一次错事，然后让孩子教育家长，并从中了解做错事不对，让家长也知道当孩子的心事，体罚不好。
－　让家长给孩子做一个好榜样，才能给孩子一个良好的生长环境。
－　感动孩子们的心灵。

边境地区的流动儿童和青少年讲述自己的生活：
－　在我的日常生活中遇到过许多的事情，但我从没有讲起过。这就像乌云遮住了太阳。参加了活动和讨论后，感觉太阳穿过云层出来了。我们开始知道一些从没想过的关于我们自己的事情。
－　我十三岁时和妈妈一起到了泰国，因为家里很困难，没有钱。那时我还小，不能帮妈妈做事，所以妈妈就带着我到泰国找工作。
－　到小学三年级我就停学了。学校很远，要走十公里的路，对一个女孩来说　　走那么远的路很困难。我父母准备盖房子，要许多钱，就决定把我送到泰国去找工作挣钱。
－　我来到中国，虽然我不想离开父母，但我很穷，人们都看不起我。到中国挣到钱后，我要回去找爸爸妈妈，我太想妈妈了。
－　村子里没有充足的水，有时我们三、四天都不能洗澡。我不喜欢这里的生活，想去其它的地方或城市。

四川省凉山州盐源县的孩子对地震的回忆：
－　地震刚开始的时候我很害怕，自家房子有很多都倒了，有很多人都在哭。
－　这里有一个小孩子在墙边玩，墙倒下来，就把小孩子打死了。
－　水把山上的石头和房子冲下来，山下的农村死了许多小孩。
－　在这房子里死了一个小孩，她是一个女的。
－　（墙倒下来）鸭和鸡都被打死了。
－　墙倒下来，把树枝都打断了。
－　人走起来，一块瓦打在头上。
-　 老师教我们可以通过玩这种游戏来学习藏语语法
- 　玩纸牌就可学语法真是太有趣了。

"以儿童为中心"的教学法主张采用开放式提问、小组活动、游戏及直观教学来提高儿童的学习兴趣，培养儿童主动学习的良好习惯，更重要的是促进儿童在教育活动中的参与。

北京流动人口子女学校的孩子们眼中的学校和生活：

－原来的学校还要带小黄帽、红领巾. 检查卫生都不用校长，值日生去检查。 这儿打防疫针太贵， 上次打有的是三十元， 有的是八十元， 正规学校才两、三块钱。 我妈妈嫌贵又怕不是正规学校， 药不管事儿。

－我妈妈说原来我在公立小学的时候字儿写得特别好老师也夸我，后来转到私立小学以后那字儿写得乱七八糟，老师什么事儿都不管。

－那所学校老师教得不认真所以就不在那里上了.除了这个原因还有那所学校要拆了。

－教室就跟这间房间差不多大， 一年级至六年级全在一间房间上。老师要给别的年级上课了就给这边的孩子留作业。

－要是当了北京人别人不会欺负你， 也不会说" 你说的是什么话呀，这不是我们北京话"

－北京人有的好有的不好。 有的吧看你太穷了，不让你进屋子，不让你和他们家小孩玩。

－北京人有什么了不起的呀，不就是北京人吗？反正都是中国人。
